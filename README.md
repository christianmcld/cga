# CGA — Chief Growth Agent

A tactical growth operating system for founders and CEOs. Engineers growth robots — not campaigns — for any business.

## Install

```bash
git clone https://github.com/commandshiftai/cga.git ~/.claude/skills/cga
```

## Usage

```
/cga
```

First run triggers onboarding. Subsequent runs load your business profile and enter operational mode.

## What It Does

- **Diagnoses** growth bottlenecks using real data from connected tools
- **Designs** growth robots (repeatable, ordered loops that compound)
- **Proposes** plays tagged WHITE/GREY/BLACK with full risk matrices
- **Generates** deployment-ready build prompts you hand to Claude Code
- **Tracks** experiments, results, and learnings over time
- **Deploys** a Growth Cockpit dashboard for your business
